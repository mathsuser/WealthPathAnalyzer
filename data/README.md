# WealthPathAnalyzer Data Directory

This folder contains all data resources used in the WealthPathAnalyzer project.

## Structure


- **raw/**  
		Contains original, unprocessed data files:
		- btc-USD_monthly.csv: Monthly Bitcoin price data.  
			Source: http://www.stooq.com
		- gold_data_2025_monthly.csv: Monthly gold price data.  
			Source: https://www.worldbank.org/en/research/commodity-markets
		- shiller_data.xls: S&P500 prices, dividends, and 10Y Treasury rates.  
			Source: Robert Shiller's dataset: http://www.econ.yale.edu/~shiller/data.htm
		- US_data_yearly.xlsx: US annual risk-free rates.  
			Source: Robert Shiller's dataset
		- (Other files may include additional CPI or rate data)

- **processed/**  
	Contains standardized and aligned datasets for analysis:
	- core_assets_master.csv: Master dataset with monthly returns and indices for core assets (S&P500, 10Y Treasury, Gold, BTC, Savings, CPI).
	- core_assets_master_metadata.json: Metadata describing columns and sources.

	> The processed data is generated by running the pipeline script: scripts/prepare_core_assets.py.

## Data Sources


The master dataset combines:
- S&P500 prices and dividends (from Shiller data)
- 10Y Treasury rates and bond returns (from Shiller and US rates)
- Gold prices (from gold monthly CSV)
- Bitcoin prices (from BTC monthly CSV)
- Savings rates (from Federal Reserve Economic Data (FRED) and Shiller data)
- Consumer Price Index (from Shiller data)
